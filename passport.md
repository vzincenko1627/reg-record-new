## 2. Описание предметной области

Краткое описание предметной области (5–7 предложений): 
Система предназначена для администраторов для управления над учетными записями пользователей. 
Система показывает список пользователей по их логину (эл. почте).
Пользователь системы может запросить обновление списка.
Администратор может производить действия с списком и учетными записями внутри списка.
Для взаимодействия с записью достаточно по ней кликнуть.
